# Zen Mode

An HDMI device that instantly switches TV content to calming videos when triggered by a remote control. Built on Raspberry Pi 5.

Codeveloping this project with my children and Claude.

## What It Does

Press a button on your TV remote â†’ TV switches from live content to a serene video (ocean waves, forests, etc.). Press again to return. Designed for family viewing situations where you want to quickly switch away from unexpected content.

## Hardware

- Raspberry Pi 5 (8GB)
- Fulfalic USB 3.0 HDMI Capture Card (4K loop-out)
- TSOP4838 IR Receiver
- Standard TV remote with colored buttons
